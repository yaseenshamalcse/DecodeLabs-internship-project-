🚀 TaskFlow - Responsive Frontend Dashboard

📌 Project Overview

TaskFlow is a responsive task management dashboard built using HTML, CSS, and JavaScript. The application allows users to manage their daily tasks efficiently through a clean and user-friendly interface. Users can add, complete, and delete tasks while tracking task statistics in real time.

This project was developed as part of a Web Development Internship to demonstrate frontend development skills, responsive design principles, and JavaScript functionality.

✨ Features

Responsive design for Mobile, Tablet, and Desktop devices
Add new tasks
Mark tasks as completed
Delete tasks
Task statistics dashboard
Dark/Light theme toggle
Local Storage support (data persists after page refresh)
Modern and user-friendly interface
🛠 Technologies Used

HTML5
CSS3
JavaScript (ES6)
Local Storage API
